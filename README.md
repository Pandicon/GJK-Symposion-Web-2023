## build instructions

### frontend

 - [Install Rust](https://www.rust-lang.org/tools/install)

### backend

 - Install JDK (17+)
 - Run `mvnw`(linux)/`mvnw.cmd`(windows) with argument `spring-boot:run` script from it's directory
